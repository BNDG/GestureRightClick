# Mac上Edge鼠标右键手势插件

Mac上的Edge一直没有内置的手势，扩展市场里也找不到crxMouse Gestures了，好像以前还经常以安全问题不让使用，于是配合gpt写了个简单的插件，
只实现了几个基本的手势。
- **双击右键**：调用系统菜单。
- **←**：后退。
- **→**：前进。
- **↑**：向上滚动一页高度。
- **↓**：向下滚动一页高度。
- **画⎤形**：刷新当前标签页。
- **画L形**：关闭当前标签页。
- **向下再向上**：将页面滚动到顶部。
- **向上再向下**：将页面滚动到底部。

## 使用
### 加载扩展程序
   1. 下载项目。
   2. Edge在地址栏输入 edge://extensions/ 并按回车键。
   3. 打开左边栏“开发者模式”开关。
   4. 点击“加载解压缩的扩展”。
   5. 选择项目的根目录进行加载。
